# My schedule App

Experimental setup for an app displaying a Cloud-user schedule. Using static mockdata.

## Requirements

This is a react native app built with [Expo](https://expo.io/). 
To run it locally you need: 

* Expo Cli

```
npm install expo-cli --global
```

* The Expo app installed on your phone
* An Expo account. Log in on both your app and on your computer.
* Be connected to the same network on both your phone and your computer.

### Run project

Clone this repository and change directory to the root folder. Install dependencies.

```
yarn
```

or 

```
npm install
```

Then run the project.

```
expo start --tunnel
```

You should now se a debug window in your browser, and the project should show in your Expo app.
